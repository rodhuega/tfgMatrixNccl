Projection Tutorial (page 5)

  • Step 5: Compute the pseudospectra on a fine grid of size 200 by 200.

    First of all, set the projection parameter back to 1. Now set the number of gridpoints to 200, and hit the `Go!' button. The computation takes me about 3 minutes, less a quarter of the predicted time without projection. The final figure is shown below, and although the lines appear jagged on the screen, they will be smooth when printed out as a postscript file. Note that the matrix dimension is now just 92, less than half of the original size.

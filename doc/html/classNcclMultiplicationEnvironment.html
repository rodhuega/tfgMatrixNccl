template<class Toperation>
class NcclMultiplicationEnvironment< Toperation >

Clase estática que contiene a las matrices y las características necesarias para realizar la múltiplicación de matrices mediante el algoritmo Summa.

Parámetros del template
Toperation,tipode la matriz(double,float)

Se construye el objeto que realizara las distitnas multiplicaciones en un entorno distribuido.

Parámetros
gpuSizeWorld,Tamañodel número de gpus que se quiere que realicen las operaciones. (Puede ser mayor que el número físico, se usarán mas veces las gpus físicas)
gpuRoot,Idde la gpu que actuará como root.
opType,Indicael tipo de operacion que se usará para las operaciones. Tiene que ser compatible con el tipo usado en la generacidad.MultDouble|MultFloat
printMatrix,indicasi se van a imprimir las matrices idA e idB en la ejecución del programa
